# Chess Habits Docs v7.1.9h

Site estático para publicar em `doc.chesshabits.app`.

## Arquivos
- `index.html`
- `documentacao-metodologica-v7.1.9h.pdf`

## Deploy recomendado na DigitalOcean App Platform
1. Envie estes arquivos para a raiz do repositório `chesshabits-docs`.
2. Crie ou atualize um componente **Static Site** conectado à branch `main`.
3. Não configure Dockerfile, build command ou run command.
4. Adicione o domínio `doc.chesshabits.app` ao componente estático.
5. Aguarde a validação DNS e a emissão automática do HTTPS.
